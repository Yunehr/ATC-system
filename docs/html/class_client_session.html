Detailed Description

Static utility class for pilot authentication.

Validates a "username,password" credential string against LoginAuth.csv and returns a structured response string consumed by the packet handler in ClientApp.

All state is held in the CSV file; no instance state is required.

std::string ClientSession::authenticate ( const std::string & credentials)
static

Authenticates a pilot against the credentials in LoginAuth.csv.

Authenticates a pilot against the credentials stored in LoginAuth.csv.

Delegates parsing to parseCredentials(), then performs a linear search of the CSV for a matching username/password pair.

Parameters
credentialsCombined credential string in "username,password" format.
Returns
One of the following response strings:
  • "Authentication Successful (License: <license>)" on success.
  • "Authentication Failed: expected Username,Password" if the credential string could not be parsed.
  • "Authentication Failed: LoginAuth.csv not found" if the data file could not be opened.
  • "Authentication Failed" if no matching record was found.

Parses credentials into a username/password pair, then performs a linear search of LoginAuth.csv for a matching row. The CSV is expected to have the following column layout (with a header row that is skipped):

Username, Password, LicenseNumber

On a successful match the pilot's license number from the CSV is included in the response string.

Parameters
credentialsCombined credential string in "username,password" format, as received from the packet payload.
Returns
One of the following response strings:
  • "Authentication Successful (License: <license>)" on success.
  • "Authentication Failed: expected Username,Password" if the credential string could not be parsed.
  • "Authentication Failed: LoginAuth.csv not found" if the data file could not be opened.
  • "Authentication Failed" if no matching record was found.
