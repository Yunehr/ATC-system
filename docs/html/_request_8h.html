Detailed Description

Defines the Request class: the application-layer command unit carried inside a pkt_req packet.

A Request is the innermost payload of the communication stack. It pairs a REQTYPE command byte with an arbitrary body buffer, and can be serialized to / deserialized from a flat byte array for embedding inside a packet.

Wire format (serialized)

[ type (1 byte) | body (bodysize bytes) ]

Typical usage

// Send side
Request req(req_weather, dataLen, dataPtr);
int32_t serialSize;
char* serial = req.serializeRequest(&serialSize);
txPacket.PopulPacket(serial, serialSize, clientId, pkt_req);
delete[] serial;
// Receive side
Request req(rxPacket.getData(), rxPacket.getPloadLength());
handleRequest(req.getType(), req.getBody(), req.getBsize());
