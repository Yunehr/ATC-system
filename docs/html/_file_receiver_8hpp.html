Detailed Description

Declaration of the FileReceiver class.

FileReceiver manages the write-side of the multi-packet file transfer protocol used to download the flight manual. It assembles incoming pkt_dat packets into a .part staging file and atomically renames it to the final path once the server signals transfer completion.

Typical usage
FileReceiver receiver;
std::string err;
if (!receiver.start("output/flightmanual.pdf", err)) { ... }
while (!receiver.isComplete()) {
packet pkt = ...; // receive from PacketTransport
if (!receiver.processPacket(pkt, err)) { ... }
}
if (!receiver.finalize(err)) { ... }
