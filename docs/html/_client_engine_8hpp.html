Detailed Description

Declaration of the ClientEngine class.

ClientEngine is the low-level networking layer of the aviation client. It owns the Winsock TCP socket and implements three request patterns that map directly onto the shared packet protocol:

Method Outgoing packet Use case
dataRequest() pkt_req + Request Weather, flight plan, telemetry
pkRequest() Raw typed packet Auth, emergency, resolve
downloadFlightManual() pkt_req + req_file Multi-packet file transfer

All methods perform CRC verification on received packets and return human-readable error strings on failure, which are forwarded to the Python UI via the stdout bridge in main.cpp.
