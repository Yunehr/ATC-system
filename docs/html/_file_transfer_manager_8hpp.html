Detailed Description

Declaration of the FileTransferManager class.

Provides static server-side data access methods for flight plans, air traffic positions, taxi/landing clearances, telemetry logging, and flight manual availability. All methods are stateless and read from or write to CSV/PDF files located in the shared data/ directory.
