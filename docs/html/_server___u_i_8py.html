Detailed Description

Tkinter-based ATC Tower Control Center dashboard.

This module provides the graphical monitoring interface for the aviation backend server. It spawns the C++ server executable as a child process, reads its stdout in a daemon thread, and presents live data across four areas:

  • A metrics row (connections, uptime, data transferred, last activity).
  • An active aircraft panel populated from real-time netstat output.
  • A system log panel (REQ-LOG-010).
  • A file transfer progress bar (REQ-SYS-070).

Server log lines are parsed for structured events (STATE:, TRANSFER START:, TRANSFER PROGRESS:, TRANSFER COMPLETE:) and dispatched to the appropriate UI handlers via root.after() so all widget updates remain on the Tk main thread.
