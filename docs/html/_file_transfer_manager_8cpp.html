Detailed Description

Implementation of the FileTransferManager class.

Provides server-side data access for flight plans, air traffic, taxi clearances, telemetry logging, and flight manual availability checks. All data files are located using a two-path probe (../data/ then data/) consistent with the rest of the server codebase.

Data files used
File Access Purpose
Flight_Plan.csv Read Flight plan lookup by flight ID
Traffic_Positions.csv Read Air traffic lookup by flight ID
StatePermissions.csv Read Taxi/landing clearance policy
SystemTrafficLog.csv Read+Append Telemetry logging
FlightManual.pdf Read (binary) Manual availability check
