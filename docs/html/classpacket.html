Detailed Description

A framed, serializable network packet with header, payload, and CRC.

The packet class handles both the transmit path (populate → serialize) and the receive path (construct from raw bytes → extract payload). It owns its internal buffers and manages their lifetime through RAII.

Transmit path

packet pkt;
pkt.PopulPacket(src, size, clientId, pkt_req);
int len;
char* wire = pkt.Serialize(&len); // wire is valid for pkt's lifetime

◆ calcCRC()

int packet::calcCRC ( )
inline

Computes the packet's checksum over all header and payload bytes.

Sums every byte of the header struct followed by every byte of the payload, treating all bytes as unsigned. This value is written into the CRC tail by Serialize() and can be compared between a transmitted and received packet to detect corruption.

Returns
The computed checksum as a signed 32-bit integer.
Note
This is a simple byte-sum checksum, not a standard CRC polynomial. It is sufficient for detecting single-byte errors but not burst errors.

int packet::PopulPacket ( char * datsrc,
int size,
char id,
char type )
inline

Populates the packet with a payload, truncating if necessary.

Copies up to MAX_PKTSIZE bytes from datsrc. If size exceeds MAX_PKTSIZE, the payload is truncated and transmit_flag is set to PKT_TRNSMT_INCOMP to signal that more packets follow. Any previously held payload buffer is freed before the new one is allocated.

Parameters
datsrcPointer to the source data to copy into the packet body.
sizeNumber of bytes to copy from datsrc.
idClient ID to embed in the header (0–15).
typePacket type to embed in the header; see PKTYPE.
Returns
The number of bytes actually consumed from datsrc. If less than size, the caller is responsible for sending the remainder in subsequent packets.

char * packet::Serialize ( int * finalsize)
inline

Serializes the packet into a flat wire buffer.

Lays out [header | payload | CRC] contiguously in an internally managed buffer. The CRC is computed over the header and payload bytes immediately before writing. Any previously held txbuff is freed first.
